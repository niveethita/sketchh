# Sketchh 🎨

A simple browser-based drawing app built with vanilla HTML, CSS and JavaScript.

## Features
- Freehand drawing on canvas
- Color picker
- Adjustable brush size
- Eraser tool
- Fill background color
- Clear canvas

## How to use
1. Pick a color and brush size
2. Draw on the canvas
3. Switch between pen and eraser using the toolbar buttons
4. Fill the background with any color using the fill button
5. Clear everything with the clear button

## Built with
- HTML5 Canvas API
- Vanilla JavaScript
- CSS
